# AutoIME

클립스튜디오, 포토샵 등 그래픽 툴에서 한글 입력 상태로 인한 단축키 씹힘 문제를 해결하는 트레이 앱입니다.

_본 프로그램은 AI의 도움을 받아 작성했습니다._

---

## 제작 배경

[륜 님의 오토핫키 한영전환 스크립트](https://iamlyun.tistory.com/entry/%EA%B0%84%ED%8E%B8-%ED%95%9C%EC%98%81%EC%A0%84%ED%99%98-%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8?category=940609)를 기반으로 제작하였습니다. 해당 스크립트는 윈도우 11 환경으로 넘어오면서 별도 설정 없이는 동작하지 않게 되었고, 이를 계기로 C++ Win32 네이티브 앱으로 새로 구현하였습니다.

> ㅁ륜ㅁ 님의 원본 스크립트는 **원사운드** 님의 포토샵 한글 단축키 스크립트를 기반으로 작성되었다고 합니다.

---

## 기능

### 자동 한영 전환

마우스가 움직일 때마다 입력 상태를 감지하여 한글이면 자동으로 영문으로 전환합니다. 말풍선 입력 등 한글이 필요한 경우 트레이 아이콘을 우클릭하여 비활성화할 수 있습니다.

### CapsLock 키 리매핑

잘 사용하지 않는 CapsLock 키를 다른 키로 대체합니다. 처음 실행 시 비활성 상태이므로, 사용하려면 트레이 아이콘을 우클릭하여 활성화해 주세요. 설정에서 바꿀 키를 변경하실 수 있습니다.

### 대상 프로그램 관리

기본으로 클립스튜디오와 포토샵이 등록되어 있으며, 트레이 메뉴 → 설정 → 대상 프로그램 관리에서 프로그램을 추가하거나 제거할 수 있습니다.

> ⚠️ **주의:** 온라인 게임을 등록할 경우 안티치트 프로그램에 감지될 수 있습니다. 용도 외 사용으로 발생하는 문제에 대해서는 책임지지 않습니다.

## 트레이 메뉴 구조

```
✓ 자동 영어 전환
  Capslock 리맵
─────────────────
  설정 ▶
    Capslock → F13 설정...
    대상 프로그램 관리...
    ─────────────────
    시작 프로그램 등록
─────────────────
종료
```

## 설정 파일

실행 파일과 같은 폴더에 `config.ini`가 자동 생성됩니다.

```ini
AutoEnglish=1
RemapEnabled=0
RemapKey=F13
Target=clipstudiopaint
Target=photoshop
```
